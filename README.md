# FASSI CIDF — Site Web Vitrine

Site web professionnel du Complexe Industriel de Détergent FASSI.

## 🚀 Démarrage rapide

```bash
npm install
npm run dev
```

## 🌐 Déploiement Vercel

```bash
npm install -g vercel
vercel deploy
```

## 🔐 Accès Admin

URL : `/admin`

## 📁 Structure

```
src/
├── components/      # Navbar, Footer
├── context/         # AdminContext (état global)
├── data/            # siteData.js (toutes les données)
├── hooks/           # useScrollReveal
└── pages/           # Home, Products, Contact, Admin...
```

## 🔧 Stack technique

- React 18 + Vite
- React Router DOM v6
- Tailwind CSS v3
- Lucide React (icônes)

## 📦 Build production

```bash
npm run build
```
